##订单接口

属性condition  int型  品相  对应关系你定吧

```java
1.查询用户所有订单（返回订单List）
localhost:2333/orderList/searchUserList?user_id=11
    
{
    "obj": [
        {
            "address": "南信大",
            "consignee": "zzyuan",
            "createDate": "2021-6-8",
            "deliverDate": "",
            "good_id": 2,
            "notes": "不要辣",
            "orderID": "1120210608225928",
            "origPrice": 79,
            "phone": "19825068798",
            "practicalPrice": 32,
            "publisher_id": 4,
            "state": 1,
            "user_id": 11
        }
    ],
    "status": 1
}

status:
	1--成功
    0--没有订单
        
        
2.增加订单
localhost:2333/orderList/addOrderList
传参：
{
    "address": "南信大",
    "consignee": "zzyuan",
    "createDate": "2021-6-9",
    "good_id": 4,
    "notes": "不要辣",
    "orderID": "1120210600231728",
    "origPrice": 79,
    "phone": "19825068798",
    "practicalPrice": 12,
    "publisher_id": 4,
    "user_id": 11
}

返回：
{
    "obj": null,
    "status": 1
}

status:
	1--成功
    0--插入失败
        
        
3.修改订单状态
localhost:2333/orderList/changeState?orderID=1120210608225928&state=2
    
{
    "obj": null,
    "status": 1
}

status：
    1--成功
    0--修改失败
    
    
state：
    0--已付款
    1--已发货
    2--交易成功
    
    
4.取消订单
localhost:2333/orderList/cancelList?orderID=1120210600231728
    
{
    "obj": null,
    "status": 1
}

status：
    1--成功
    0--删除失败
```
